STEP 1: UPDATE FRONTEND (5 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

In CosmicOS.jsx, find and replace TWO fetch calls (lines 1064 & 1510):

FROM:
  const resp = await fetch("https://api.anthropic.com/v1/messages", {

TO:
  const resp = await fetch("/api/oracle", {

Also update request body (around line 1067):

FROM:
  body: JSON.stringify({
    model:"claude-sonnet-4-20250514",
    max_tokens:1000,
    system:systemPrompt,
    messages:[...history,{role:"user",content:text}],
  }),

TO:
  body: JSON.stringify({
    message: text,
    systemPrompt: systemPrompt,
    history: messages.slice(-10).map(m=>({role:m.role,content:m.content})),
    mode: mode,
  }),

And response parsing (around line 1075):

FROM:
  const reply = data.content?.[0]?.text || "...";

TO:
  const reply = data.reply || data.content?.[0]?.text || "...";

STEP 4: SET API KEY (2 min)

1. Vercel Dashboard → Your Project
2. Settings → Environment Variables
3. Add:
   Name:  ANTHROPIC_API_KEY
   Value: sk-ant-... (get from https://console.anthropic.com)
4. Save

⚠️  CRITICAL: Never hardcode API keys in code!
